## 🚀 About the Project

Redux Toolkit (RTK) is the official, recommended way to write Redux logic. It simplifies state management by reducing boilerplate code and providing powerful utilities for creating slices, reducers, and asynchronous actions.

This repository contains examples and practice projects to help developers understand how Redux Toolkit works in real-world React applications.

---

## ✨ Features

* Modern Redux Toolkit setup
* Store configuration
* Creating and managing slices
* State management with `createSlice`
* Dispatching actions
* Reading state using `useSelector`
* Updating state with `useDispatch`
* Redux DevTools integration
* Clean and scalable project structure
* Best practices for Redux Toolkit

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/prabhatrana666/redux-toolkit-project.git
```

Navigate to the project:

```bash
cd redux-toolkit-project
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
